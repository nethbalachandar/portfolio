This project explores data-driven methods for yield improvement in semiconductor manufacturing using the SECOM dataset with 1,567 samples and 590 sensor features. Through effective data preprocessing, feature selection using mutual information, and XGBoost modeling, we identified key process parameters that influence yield outcomes, providing actionable insights for process control and optimization.

# Convert labels from {-1, 1} to {0, 1}
data['Label'] = data['Label'].map({-1: 0, 1: 1})
print("Updated Label Distribution:\n", data['Label'].value_counts())

# Separate features and labels
X = data.drop('Label', axis=1)
y = data['Label']

# Standardize features
scaler = StandardScaler()

X_scaled = pd.DataFrame(X_scaled, columns=X.columns)

# Train-test split
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y,
