Graph Neural Networks for Materials Selection

Leveraging GNNs for intelligent materials discovery

← Back to Portfolio

Overview

This project explores the use of Graph Neural Networks (GNNs) to predict material properties and support data-driven materials selection.

Problem Statement

Material discovery is slow and expensive. GNNs help model crystalline structures directly and predict their properties efficiently.

Approach

  1. Data Source: Materials Project database via pymatgen
  2. Graph Construction: Crystal structure graphs using pymatgen + matminer
  3. Modeling: PyTorch Geometric + MPNN
  4. Evaluation: MAE, R² on test set

Tools & Technologies

  • Python, PyTorch Geometric, NumPy
  • pymatgen, matminer, ASE
  • Jupyter Notebook
  • Materials Project API

Results

  • MAE: 0.047 eV/atom
  • R² Score: 0.92
  • Generalizes well to new crystal systems

Applications

  • Semiconductors, catalysis, battery design
  • Interpretability with attention maps
  • Integrates into high-throughput pipelines
