Wafer Defect Detection Using Convolutional Neural Networks

← Back to Portfolio

🔍 Overview

This project applies convolutional neural networks (CNNs) to identify and localize defects in wafer images, simulating a semiconductor manufacturing quality control pipeline.

🧩 Problem Statement

Defects in semiconductor wafers can significantly impact device yield and reliability. Manual inspection is time-consuming and error-prone. This project explores automated image-based defect detection using deep learning.

🛠️ Tools & Technologies Used

  • Python, OpenCV, NumPy, Matplotlib
  • Keras + TensorFlow backend
  • Custom dataset (mock wafer defect images)
  • Jupyter Notebook for experimentation

🧠 How It Works: A to Z

  1. Data Collection: Gathered and simulated wafer defect image data using open-source examples and augmentation.
  2. Preprocessing: Normalized, resized (128x128), and augmented data using flipping, noise injection, and rotation.
  3. Model Architecture: Built a 3-layer CNN with ReLU activation, dropout, and max-pooling. Output layer used softmax for multi-class defect classification.
  4. Training: Used categorical crossentropy loss, Adam optimizer, 20 epochs, and an 80:20 train-test split. Achieved ~92% test accuracy.
  5. Defect Localization: Overlaid detected regions using bounding boxes with OpenCV.
