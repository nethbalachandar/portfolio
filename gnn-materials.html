Graph Neural Networks for Materials Selection

← Back to Portfolio

🔍 Overview

This project explores the use of Graph Neural Networks (GNNs) to predict material properties and support data-driven materials selection. GNNs capture structural relationships between atoms and bonds to model physical behavior in materials science.

📌 Problem Statement

Material discovery and optimization are bottlenecked by costly experiments and simulations. Classical ML struggles to model crystalline structures. GNNs offer a way to directly learn from atomic graphs to predict mechanical, electronic, or thermodynamic properties.

🧠 Approach

  1. Data Source: Used the Materials Project database (via pymatgen) to extract structural and property data.
  2. Graph Construction: Built atom-bond graphs from crystal structures using pymatgen and matminer.
  3. Modeling: Used PyTorch Geometric to train a GNN (based on Message Passing Neural Network architecture) to predict properties such as formation energy and band gap.
  4. Evaluation: Used MAE and R² to assess regression performance on a held-out test set.

🛠️ Tools & Technologies

  • Python, PyTorch Geometric, NumPy, pandas
  • pymatgen, matminer, ASE
  • Jupyter Notebook
  • Materials Project API

📊 Results

The GNN model achieved:

  • MAE: 0.047 eV/atom on formation energy
  • R² Score: 0.92
  • Consistent generalization to unseen crystal systems

🌍 Applications

  • Accelerated screening of materials for semiconductors, catalysis, and batteries
  • Improved interpretability through atom-level attention maps
  • Scalable integration with high-throughput materials pipelines
