Yield Optimization in Semiconductor Manufacturing

A Data-Driven Approach using XGBoost and Mutual Information

Project Overview

  • 590 sensor features from SECOM dataset
  • Severe class imbalance: 93% fails / 7% passes
  • Missing values handled using median imputation

This project leverages feature selection and XGBoost modeling to classify pass/fail labels in semiconductor wafers based on sensor data. The work emphasizes effective preprocessing and the importance of key sensors in yield outcome.

Data Preprocessing

Median imputation was used due to outlier robustness. Features were standardized with StandardScaler. Zero-variance features were dropped (116 removed).

Mutual Information was used to select the top 50 features due to its effectiveness with non-linear dependencies.

Deployment

This model can be deployed in semiconductor fabs for real-time monitoring of critical sensor values. Threshold-based alerts can be integrated to enable proactive yield improvements.
